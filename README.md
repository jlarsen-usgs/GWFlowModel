# GWFlowModel
A Python project to focused on creating a Groundwater Flow Modeling System. Groundwater Flow equations based are based off of MODFLOW. 

## Vision
The project vision is to develop an easily extendable python (or maybe cross compiled with C++ or Fortran) code base that is able to 
simulate groundwater flow. The modeling code should be validated against existing benchmarks and test cases in the literature. 

## Purpose
This project is currently for personal enrichment
